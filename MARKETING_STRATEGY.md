타로냥은 기술적으로 완성(8 페이지 · AI 리딩 · 인증 · 결제)되었으나 유기적 트래픽이 거의 없습니다.

## 1. Naver SEO 최적화 가이드

### 1.1 Naver 검색 알고리즘 특성 (Google vs Naver)

| 항목 | Google | **Naver** |
|------|--------|-----------|
| 크롤러 | Googlebot | **Yeti** (`robots.txt`에 이미 허용됨 ✅) |
| 색인 방식 | 자연스러운 링크 발견 | **서치어드바이저 수동 RSS/사이트맵 제출 필수** |
| 순위 요인 | 링크 권위, 콘텐츠 품질 | **자체 플랫폼 우선(블로그·카페), 키워드 매칭, 클릭률** |
| 신규 사이트 | 점진적 색인 | **등록하지 않으면 수개월간 미노출** |
| 스니펫 | 구조화데이터 | **구조화데이터(JSON-LD) 부분 지원** |

> **핵심 차이:** Naver는 "서치어드바이저 등록 + RSS 제출"을 하지 않으면 사실상 검색에 노출되지 않습니다. Google은 자연 색인되지만 Naver는 능동 제출이 필수입니다.

### 1.2 Naver 서치어드바이저 등록 절차 (CTO 실행 가이드)

> ✅ = 이미 완료된 항목 / ⬜ = 실행 필요

#### Step 1: 사이트 소유 확인
1. https://searchadvisor.naver.com/ 접속 → 네이버 계정 로그인 (업무용 계정 권장)
2. **"웹마스터 도구"** → **"사이트 추가"** → `https://taronyang.com` 입력
3. 소유 확인 방법 선택:
   - **방법 A (권장): HTML 파일 업로드** — 제공된 `naver_xxx.html` 파일을 `frontend/` 루트에 배치
   - 방법 B: HTML 메타태그 — `<meta name="naver-site-verification" content="xxx" />` 를 모든 페이지 `<head>`에 추가
   - 방법 C: DNS — `taronyang.com` DNS에 TXT 레코드 추가
4. 확인 완료 후 사이트가 "정상" 상태로 표시되는지 확인

#### Step 2: RSS 피드 제출 ⬜
- **서치어드바이저 → RSS** 메뉴 → `https://taronyang.com/rss.xml` 등록
- ✅ RSS 피드 파일이 이미 존재함 (`frontend/rss.xml`)
- 등록 후 "요청 성공" 확인. Naver가 주기적으로 RSS를 크롤링하여 신규 콘텐츠를 색인

#### Step 3: 사이트맵 제출 ⬜
- **서치어드바이저 → 사이트맵** 메뉴 → `https://taronyang.com/sitemap.xml` 등록
- ✅ 사이트맵 파일이 이미 존재함
- ⚠️ **카드 페이지 22개 URL을 sitemap.xml에 추가해야 함** (현재 4개 URL만 등록)
  - `generate-card-pages.mjs`가 이미 sitemap 통합 기능을 포함하므로, 빌드 시 자동 추가됨

#### Step 4: robots.txt 확인 ✅
- ✅ `robots.txt`에 `Yeti` (Naver 크롤러) 명시적 허용 완료

#### Step 5: 수동 URL 제출 (초기 부스팅)
- **서치어드바이저 → 요청 → 웹 페이지 수집** → 주요 URL 개별 제출:
  - `https://taronyang.com/` (홈)
  - `https://taronyang.com/tarot` (타로 리딩)
  - `https://taronyang.com/daily` (오늘의 운세)
  - `https://taronyang.com/cards/` (메이저 아르카나 인덱스)
  - 카드 상세 페이지 22개

#### Step 6: 모니터링
- 서치어드바이저 대시보드에서 주간 단위로 "노출수 · 클릭수 · 노출 순위" 확인
- "색인 현황" 탭에서 차단된 URL이 없는지 점검

| 순위 | 키워드 | 월간 검색량(추정) | 경쟁도 | 타로냥 랜딩 페이지 |

### 1.4 네이버 블로그 vs 웹사이트 SEO

Naver는 자체 플랫폼(블로그·카페)에 가중치를 줍니다. **두 가지를 병행**하는 것이 최적입니다.

| 전략 | 웹사이트(taronyang.com) | 네이버 블로그 |
|------|------------------------|---------------|
| 역할 | 메인 서비스, 트래픽 전환 | 검색 노출 확대, 역링크 |
| 콘텐츠 | 카드 해설, 운세 (✅구축됨) | 블로그 포스트(아래 10선) |
| 장점 | 전환율, 브랜드 통제 | 빠른 Naver 색인, 상위 노출 |
| 단점 | Naver 색인 지연 | 전환율 낮음 |

**실행:** 네이버 블로그 "타로냥" 개설 → 주 2회 포스팅 → 각 포스트에서 `taronyang.com`으로 링크. 블로그는 미러링이 아닌 **요약+링크** 형태로 작성 (중복 콘텐츠 페널티 방지).

### 4.2 커뮤니티 진출 가이드라인 (스팸 방지)

> ⚠️ **절대 규칙:** 어디든 첫 글에 서비스 링크를 직접 홍보하지 마세요. 먼저 가치를 제공하고 신뢰를 쌓은 후 자연스럽게 언급합니다.

#### 네이버 카페
- 타로 관련 대형 카페 3~5곳 가입 (예: 타로 카페, 운세 카페)
- **2주간:** 질문에 성실히 답변 (타로 해석 지식 보여주기)
- **3주차부터:** "제가 만든 사이트에서 무료로 타로 볼 수 있어요" 자연스럽게 언급
- 카페 규정 확인: 일부 카페는 외부 링크 금지

#### 디시인사이드 타로갤러리
- 매일 글 중 질문글 3~5개에 타로 해석 답변
- 시그니처나 닉네임에 "타로냥" 뉘앙스 (직접 링크 X)
- 답변 품질로 팬덤 형성 → 자발적 유입

#### 인스타그램 / 틱톡
- **인스타:** 카드 해설 캐러셀 (이미지 5~8장) — 78장 시리즈화
- **틱톡:** "오늘의 타로 1장 뽑기" 숏폼 (15초) + 프로필 링크
- 해시태그: `#타로 #무료타로 #오늘의운세 #타로카드 #AI타로`
- 주 3회 업로드로 알고리즘 가중치 확보

#### Reddit (영문 시장 테스트)
- r/tarot (1.2M 구독), r/astrology — 영문 콘텐츠 필요
- 타로냥 영문 버전이 있다면 진출 가치 있음. 현재는 한국어 서비스이므로 **우선순위 낮음**
- 차후 글로벌 확장 시 재검토

|--------|------|-----------|-----------------|-------------|

### 5.2 경쟁사 전략 역추적 & 대응

1. **타로마루의 Naver 상위 노출 비결:** 대량의 네이버블로그 포스팅 + 카드별 상세 페이지
   - → 타로냥도 22장 카드 해설(✅) + 블로그 포스팅으로 대응
2. **타로유니의 인스타 활용:** 카드 이미지 콘텐츠로 바이럴
   - → 타로냥 카드 심볼(이모지 기반)을 시각 콘텐츠로 활용
3. **썬앤문의 카페 마케팅:** 네이버 카페 오가닉 홍보
