# 트위터/X 스레드 — 타로냥 소개 + 실제 리딩 결과 공유

> **용도:** 트위터/X 스레드 (연쇄 트윗)
> **톤:** 캐주얼, 정보 밀도 높음, 살짝 감성, 스레드 마스터 스타일
> **목표:** 스레드 전체 읽기 → 마지막 트윗에서 링크 클릭
> **타겟:** 타로/운세/점성술 관심 한국 트위터 유저

---

## 스레드 개요

| 항목 | 내용 |
|------|------|
| 트윗 수 | 7개 (첫 트윗 + 후속 6개) |
| 미디어 | 카드 이미지, 리딩 결과 스크린샷 2~3장 |
| 핵심 | "무료인데 해석이 깊다" + 실제 결과로 증명 |
| CTA | 마지막 트윗에 링크 |

## 트윗 1/7 (헤드 — 훅)
```
무료 AI 타로 사이트 찾았는데 해석 퀄리티가 미쳐서 공유 🔮

일반 무료 사이트는 카드 2줄 설명하고 끝인데
얘는 카드 하나당 한 문단으로 풀어주고
추가 질문도 대화로 할 수 있음

직접 뽑은 결과로 보여줄게 — 쓰레드 👇
```
**미디어:** (없음, 텍스트 훅)

## 트윗 2/7 (소개 + 어떻게 작동하는지)
```
타로냥(타로+고양이냥)이라는 곳이야

1️⃣ 주제 선택: 연애운 / 재물운 / 직장운 / 종합운 / 꿈해몽
2️⃣ 카드 3장 뽑기 (과거-현재-미래)
3️⃣ AI가 각 카드를 위치에 맞게 디테일 해석
4️⃣ 궁금한 거 추가로 물어보기 가능 (이게 핵심)

고양이 캐릭터가 "~냥" 하면서 친절하게 알려줌 🐱
```
**미디어:** 카테고리 선택 화면 스크린샷

```

과거 👁️ 여사제 (정방향)
현재 💕 연인 (정방향)
미래 ☀️ 태양 (정방향)

3장 다 정방향으로 나와서 일단 기분 좋았고...
여기서부터 해석이 소름이었음
```
**미디어:** 뽑힌 3장 카드 결과 화면

## 트윗 5/7 (실제 결과 공유 3 — 조언)
```
현재 연인:
"중요한 선택의 순간. 진지한 감정으로 발전할 가능성"

미래 태양:
"밝고 긍정적인 결과. 솔직하게 마음 표현하면 좋은 결과가"

→ 즉, "지금이 타이밍이니까 용기 내서 다가가라"는 해석

막연히 고민만 하다가 카드 해석 보니까
"아, 지금 움직여야 하는구나" 하고 정리됨
이게 타로 보는 진짜 가치 같음
```
**미디어:** 종합 조언 부분 스크린샷

## 운영 가이드

### 업로드 팁
1. **헤드 트윗이 전부** — 1/7 트윗의 클릭률이 스레드 성공을 결정
2. **첫 15분** — 헤드 트윗의 좋아요/리트윗이 많아야 알고리즘 노출
3. **미디어 필수** — 텍스트만 있는 트윗보다 이미지 첨부 시 참여율 3배
4. **해시태그는 첫 트윗에만** — #타로 #무료타로 #연애운 (너무 많으면 스팸 인식)
5. **트윗 간격 2~3초** — 스레드로 자동 인식되도록 연속 게시

### 인터랙션 전략
- **답글 오면 즉시 응답** — "어떤 카드 나왔어?" → 적극 해석해줌
- **리트윗 유도** — "본인 결과 quote tweet으로 공유해줘" → 바이럴 확대
- **24시간 후 인사이트 트윗 추가** — "어제 스레드에 반응 많아서 추가 정리"로 리치 확장

### 변형 버전
- **직장운 버전** — 퇴사 고민하는 사람 타겟
- **재물운 버전** — 투자/부업 관심층 타겟
- **꿈해몽 버전** — 꿈 해석 호기심 자극

## ⚠️ 주의사항
- X(트위터)는 한국 타로 커뮤니티가 작으므로 기대 트래픽은 낮지만, 인플루언서 공유 시 폭발적 확산 가능
- 링크 트윗은 X 알고리즘이 도달을 낮출 수 있음 → 마지막 트윗(7/7)에만 링크, 이전 트윗은 순수 콘텐츠
- "광고 아니에요"라고 방어하기보다 콘텐츠 자체로 가치 증명
